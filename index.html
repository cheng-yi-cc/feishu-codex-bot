快速接入你的机器

三步完成本地 Windows 开发机的部署与自启动

powershell 
# 1. 确保已安装 Node.js 和 codex cli
codex login status

# 2. 克隆仓库并安装依赖
git clone https://github.com/cheng-yi-cc/feishu-codex-bot.git
cd feishu-codex-bot
npm install

# 3. 复制环境变量并按文档配置飞书密钥
Copy-Item .env.example .env

# 4. 生产配置与跑通健康检查
npm run typecheck && npm test && npm run build
npm run start

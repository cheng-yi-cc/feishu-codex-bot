01 // 核心功能模块
001

自然语言命令直达

跳过繁琐的 GUI 交互。只需在飞书中发送自然语言，即可直达本地工作区。更支持配合严格的 /cwd 或 /run 命令进行精确制导。

002

持久化上下文显存

基于 SQLite 驱动的状态记忆。它能精确记住你的开发目录、分支状态和崩溃日志。随时可用 /resume 命令无缝恢复连接交互。

003

串行任务调度锁

配备严格队列机制的任务编排系统。完美解决多线程覆盖与资源竞争锁定问题，防止 AI 误操作导致你的本地文件系统并发写坏。

004

深层多模态附件管道

支持在飞书直接拖拽报错截图与日志文件。系统将附件安全拉取至本地沙盒环境，全自动交由底层视觉与大模型进行即时技术分析。

02 // 架构执行流
INPUT
FEISHU_IM

通过长连接 WebSocket 瞬时获取你的指令与多模态附件。

ORCHESTRATE
LOCAL_BOT

解析飞书意图、还原 SQLite 历史上下文、严格调度安全沙箱。

EXECUTION
CODEX_CLI

接管工作区 Root 权限，自动调用大模型修改源码并稳定拉起回归测试。

03 // 引导启动

部署本地实例

分秒之间在您的 Windows 机器上完成底层环境的部署拉起。环境前置要求：完成 Node.js 与 Codex CLI 工具侧的装载。

  • ► 重定向克隆远程仓库

# 密钥配置覆写

# 编译挂载并执行生命周期
